Обновления

Публичные release notes по важным сервисным и контентным изменениям.

2026-02-19

  • Мультиязычный релиз: доступно 11 языков интерфейса.
  • Запущен этот standalone GitHub Pages docs-сайт.
  • Добавлены страницы: Как это работает, Статус, Цены, Приватность, FAQ и Обновления.
  • Добавлены robots.txt и sitemap.xml для индексации.

2026-02-11

  • Память контекста сокращена до 2 последних сообщений для снижения API-стоимости.
  • Бесплатная квота переключена на 5 Quick-запросов в день для подписчиков.

2026-02-07

  • Итоги первой недели: 75 пользователей открыли и попробовали бота.
  • Итоги первой недели: 50 пользователей подписались на канал.
  • Итоги первой недели: обработано 3000+ вопросов.
  • Итоги первой недели: около $17 расходов на API.
  • Опубликован cost-анализ длинных чатов и роста трат токенов.
  • Выявлено, что полный контекст треда увеличивает стоимость запроса в длинных диалогах.
  • Предложено ограничить окно памяти недавними сообщениями как компромисс по доступности.

2026-02-06

  • Опубликован roadmap по расширению multi-mode.
  • Анонсирована линейка режимов: Quick (default), Smart, Internet, Images.
  • Анонсирована модель монетизации новых платных режимов с ограниченным free-использованием.

2026-02-05

  • Бот отвечает прямо на сообщения пользователя для более чистого chat flow.
  • Бот показывает статус набора, пока Grok генерирует ответ.

2026-02-02

  • Добавлены более богатые форматированные ответы для лучшей читаемости.
  • Улучшен structured output для длинных текстов и code-heavy ответов.

2026-01-30

  • Релиз минимально жизнеспособной версии бота.
  • На старте память контекста отсутствовала, история сообщений не сохранялась.
  • В стартовом релизе добавлена английская локализация.
  • Доступ к боту на первом этапе требовал подписки на канал.

2026-01 — 2026-02 (ранние обновления)

  • Добавлена кнопка New Chat для сброса текущего контекста диалога.
  • Добавлен переключатель режимов Grok: Quick, Smart, Internet, Images.
  • Режим Internet дал web-search и fact-check сценарии.
  • Опубликованы пакеты пополнения токенов в платежном меню бота.
