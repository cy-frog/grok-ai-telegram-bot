Changelog

Public release notes for significant service and content updates.

2026-02-19

  • Multilingual release: 11 interface languages are now available.
  • Published this standalone GitHub Pages docs site.
  • Added dedicated pages: How It Works, Status, Pricing, Privacy, FAQ, and Changelog.
  • Added robots.txt and sitemap.xml for indexing.

2026-02-11

  • Context memory reduced to last 2 messages to lower API costs.
  • Free quota switched to 5 Quick requests per day for subscribers.

2026-02-07

  • Week-1 milestone: 75 users opened and tried the bot.
  • Week-1 milestone: 50 users subscribed to the channel.
  • Week-1 milestone: 3000+ questions were processed.
  • Week-1 milestone: about $17 spent on API costs.
  • Published cost-analysis update on long chats and token spend growth.
  • Identified that full-thread context causes increasing per-request cost in long conversations.
  • Proposed limiting memory window to recent messages as an affordability trade-off.

2026-02-06

  • Published roadmap update for multi-mode expansion.
  • Announced mode lineup: Quick (default), Smart, Internet, and Images.
  • Announced monetization approach for new paid modes with limited free usage.

2026-02-05

  • Bot now replies directly to user messages for cleaner chat flow.
  • Bot now shows typing status while Grok is generating a response.

2026-02-02

  • Shipped richer formatted responses for better readability.
  • Improved structured output for long text and code-heavy answers.

2026-01-30

  • Released the minimum viable version of the bot.
  • At launch, context memory was not available and message history was not remembered.
  • English localization was added in the initial release.
  • Bot access required channel subscription during the initial rollout.

2026-01 to 2026-02 (earlier updates)

  • Added New Chat button to reset current conversation context.
  • Added Grok mode switcher: Quick, Smart, Internet, and Images.
  • Internet mode enabled web search and fact-check workflows.
  • Published token top-up packs in the in-bot payment menu.
